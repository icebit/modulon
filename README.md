# Modulos v0.1.6 Buttered Potato

Modulos is a 64 bit operating system written in Rust.

# Features

* 64 bit
* VGA module
* Physical page frame allocator
* Panic handling

Version Buttered Potato: Slightly more functional than a potato loaded onto a ROM.
